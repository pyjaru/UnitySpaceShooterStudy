# Space Shooter - Unity 튜토리얼 따라하기

- 유니티의 튜토리얼을 따라한 프로젝트입니다.  
  (일부 코드 및 설정값은 튜토리얼 영상과 100% 일치하지 않습니다.)
- 깃헙 저장소에서 가져온 에셋 데이터에서 문제가 발생한 경우에는 에셋 스토어에서  
Unity에서 업로드한 Space Shooter 에셋을 다운로드 후 임포트를 합니다.
